Lists

The lists are the listboxes just below the Command: entry. By default, there are three lists:
  • Procs contains a list of all the Tcl procedures in the target application.
  • Globals contains a list of the global variables of the target application.
  • Windows contains a list of the windows of the target application.
Each list also has a menu in the menu bar. The menu contains a set of standard operations, followed by list-specific operations (if any are available.) The standard list operations are:
  • Show a X.. brings up a dialog that prompts for the name of a proc, global, or window and places the its value in the value window.
  • Find X... brings up a dialog that searches the list.
  • Edit Filter... brings up a dialog that allows editing of the list filter. The list will only displays those items that either match or don't match a set of patterns. (The default list of patterns varies for each list type.)
  • Remove List removes the list from the tkinspect window. The list can be added back again via the Options menu.
